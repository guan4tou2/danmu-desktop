# Danmu Desktop v5.3.1

Patch release for portable-only desktop packaging. Runtime behavior remains aligned
with v5.3.0: the desktop client is finalized, the viewer/server path uses the
unified WebSocket runtime, and admin cleanup work remains included.

## Downloads

- Windows: portable x64 `.exe`
- macOS: arm64 `.zip`
- Linux: AppImage and `.deb`

Windows installer output is intentionally not published. macOS DMG output is
also intentionally not published.

## macOS Signing Status

macOS builds remain ad-hoc signed and are not notarized. This is the current
expected state. If Gatekeeper blocks first launch after extracting the ZIP, run:

```bash
sudo xattr -r -d com.apple.quarantine 'Danmu Desktop.app'
```

## Verification

- Release config regression tests cover Windows portable-only and macOS ZIP-only
  targets.
- Desktop test suite covers tray, preferences/about, onboarding removal, renderer,
  overlay, settings, and auto-update UI wiring.
- Local macOS package verification confirms `v5.3.1` produces a ZIP package and
  `latest-mac.yml` points at the ZIP asset.

## Notes

- Formal signing and notarization are unchanged.
- Windows real-machine UI/Overlay smoke is planned for a Windows host before the
  Windows package is treated as fully smoke-tested on hardware.
